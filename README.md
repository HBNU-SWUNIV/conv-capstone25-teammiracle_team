## <u>Teamate</u> Project Background
- ### 필요성
  - 자율주행 기술이 발전하면서 장애물 인식 및 회피 성능 향상이 필수적이다.
  - 기존의 라이다(LiDAR) 기반 시스템은 고가의 센서 의존성과 악천후에서의 한계가 있다.
  - 머신러닝을 활용한 카메라 기반 인식 기술을 통해 저비용이면서도 정밀한 자율주행을 구현할 필요가 있다.
    
- ### 기존 해결책의 문제점
  - 기존 센서 기반 방식은 비용 부담이 크고 환경 변화(빛, 반사, 날씨)에 민감하다.
  - 단순 이미지 처리 방식은 복잡한 도로 환경에서의 실시간 인식·회피에 한계가 있다.
  
## System Design
  - ### System Requirements
    - 장애물 인식(Object Detection) : 도로 환경에서 발생 가능한 다양한 장애물을 머신러닝 모델로 탐지 및 분류
    - 회피 경로 생성(Path Planning) : 인식된 장애물을 기반으로 실시간 회피 경로를 생성하는 제어 알고리즘 개발
    - 데이터셋 구축(Data Collection) : 주행 환경에서 수집된 영상 데이터를 정제하여 학습용 데이터셋으로 구축
    
## Case Study
  - ### Description
    - Python 및 OpenCV 기반의 이미지 데이터셋 구축
    - 실시간 장애물 회피 알고리즘 개발 및 시뮬레이션 검증
  
  
## Conclusion
  - ### 기술적 성과
    - 머신러닝 기반의 저비용 자율주행 장애물 인식 및 회피 시스템 구현
  
  - ### 사회적 효과
    - 자율주행 기술의 대중화 및 산업 경쟁력 향상에 기여
